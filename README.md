# Flask Task Manager with Login System

A simple intermediate Python web project built using **Flask**, **SQLite**, and **Docker**.  
This application allows users to register, log in, manage tasks, and track task completion.

## Features

- User registration
- User login and logout
- Password hashing using Werkzeug
- Add new tasks
- Mark tasks as completed
- Delete tasks
- SQLite database integration
- Docker support for easy setup and running

## Tech Stack

- Python
- Flask
- SQLite
- Werkzeug
- Docker
